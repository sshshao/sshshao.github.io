## Java

### Regio Vinco Map Editor
[Software Design Description (PDF)](https://www.google.com)
Regio Vinco is a map-picking game made by the professor. The Regio Vinco Map Editor is a editing software that can construct maps for the game. The editor use Simple App Framework to perform basic file operations. The editing file and exported map is constructed in JSON format.

This software is the first "big" project with 3000+ lines of code I have done, and which started from software design stage.
